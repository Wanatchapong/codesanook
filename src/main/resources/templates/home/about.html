How 's codesanook.com happened?

ในโลกยุคปัจจุบัน เราถูกรายล้อมด้วย technology ต่างๆ มากมาย ไม่ว่าจะเป็นโทรศัพท์มือถือ computer เครื่องใช้ฟ้า ยานพาหนะต่างๆ โดยที่อุปกรณ์ทุกชนิดเหล่านี้ ล้วนมีสิ่งที่เปรียบเหมือนสมองกล เป็นชุดคำสั่งให้อุปกรณ์เหล่านี้ทำงานได้ โดยมีผู้ที่สร้างคำสั่งนี้ และใส่เข้าไปในอุปกรณ์นั้นๆ คนเหล่านี้อาจจะมีชื่อเรียกต่างกันในแต่ละบริบทย่อยๆ ไม่ว่าจะเป็น developer, programmer, software engineer แต่ผมขอเรียกรวมๆ กันว่า software engineer แล้วกันนะครับ เพราะผมมองว่าคนกลุ่มนี้เป็นนักสร้างสรรค์ ผู้สร้างนวัตกรรมใหม่ๆ

ด้วยเหตุนี้ วงการธุรกิจจึงมีความต้องการ software engineer เป็นอย่างสูง เพื่อสร้างสรรค์ผลิตภัณฑ์ใหม่ๆ ตอบสนองความต้องการของผู้บริโภค และช่วยชีวิตของคนให้ดีขึ้น
ดังจะเห็นได้จากบริษัทที่สำคัญของโลก ไม่ว่าจะเป็น Microsoft, Google, Apple, Facebook ล้วนถูกขับเคลื่อนจากกลุ่ม software engineer กลายเป็นผลิตภัณฑ์หรือ software ให้เราใช้งาน ไม่ว่าจะเป็น Windows, google.com, Android, iPhone, facebook.com แต่ประเด็นสำคัญที่หลีกเลี่ยงไม่ได้ก็คือ "ประเทศไทยขาดแคลน software engineer อย่างมาก" โดยเฉพาะอย่างยิ่งธุรกิจ startup ที่ไม่สามารถจ่ายเงินเดือนสูงๆ ได้ แต่มีความฝันและพลังอย่างแรงกล้าที่อยากสร้าง software ของคนไทยออกสู่สายตาชาวโลก

codesanook.com จึงเกิดขึ้นด้วย passion ส่วนตัวของผม ที่
  • ผมมีความสุขเมื่อได้สร้างเมื่อได้เขียน program แล้วเห็นผลลัพธ์ทันทีว่ามันทำงานได้
  • ความสุข ความตื่นเต้นเมื่อผมได้ค้นพบ technology ใหม่ๆ และนำมาใช้ในงานได้อย่างมีประโยชน์
  • ความสุข เมื่ออ่านหนังสือ แล้วเข้าใจที่มาที่ไปของ technology นั้นๆ
  • ความสุข ที่ได้สอนได้และแบ่งปันความรู้กับเพื่อนๆ ในวงการเดียวกัน
  • ความสุขเมื่อเห็นคนมีความสุขกับสิ่งที่เราได้สร้าง
codesanook.com ต้องการเป็นพื้นที่ ที่ช่วยส่งเสริม eco system ที่สร้างสรรค์ของวงการ software engineer ไทย
  • ให้ software engineer ไทย ได้เข้ามาแบ่งปันความรู้ ประสบการณ์
  • ได้เข้ามาถามตอบปัญหาข้อสงสัย
  • ให้คำปรึกษาน้องๆ นักศึกษาในการพัฒนาความรู้ทักษะของตนเองด้าน programming
  • เปิดโอกาสให้กับเด็กๆ เยาวชนคนรุ่นใหม่ในต่างจังหวัดได้รู้จักว่า programming คืออะไร เป็นอย่างไร
  • ส่งเสริมให้เยาวชน เริ่ม programming ได้ตั้งแต่ประถม มัธยม
  • สร้าง software engineer ไทยให้มีความเชี่ยวชาญ แข็งแกร่ง และสง่าในเวทีโลก
  • สร้างทัศนคติที่ถูกต้องกับการ programming ว่าไม่ได้เป็นสิ่งที่ยากเกินไป แต่เป็นสิ่งที่สนุก ตื่นเต้น ท้าทาย
  • ส่งเสริมพลังแห่งความคิดเชิงบวก
  • ช่วยผลักดันคุณภาพการศึกษาไทย

ความฝัน หรือความตั้งใจนี้ คงไม่สามารถสำเร็จได้แค่ผมคนเดียว แต่ต้องจากความร่วมมือของพี่น้อง software engineer ไทยทุกๆ คน ในการเข้ามาแบ่งปันกันที่ codesanook.com ผมเชื่อเสมอว่า "เราทำได้"
